# Jogo da Velha 🎮 - Equipe 5
App simples, com o objetivo de colocar em prática Lógica de Programação, e boas maneiras de identação no Python.

## 🚀 Sobre o Projeto
O programa consiste no jogo clássico da velha, entre dois jogadores, sendo executado diretamente pelo terminal.

### 🛠️ Funcionalidades
- **Entrada de Dados:** Validação de escolhas de linhas e colunas.
- **Alternância de Turnos:** Gerenciamento automático entre o Jogador 1 (X) e Jogador 2 (O).
- **Verificação de Condições:** - Identifica vitória em linhas, colunas ou diagonais.
  - Detecta empate (velha) quando o tabuleiro está cheio.

## 💻 Tecnologias Utilizadas
- **Linguagem:** Python 3.x


## Conceitos aplicados
- Listas, loops (`while`, `for`), condicionais e funções.


## 📂 Como executar
1. Instale o Python em sua máquina
2. Faça o download do repositório
3. Abra-o em uma IDE ou pelo VS Code
